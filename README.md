## Vi tips

- `ZZ` - Save and exit quickly
- `dd` - Delete entiry line
- Press `Esc`, then type `undo` - to undo recent change
- Press `Esc`, then `SHIFT + V` - to select multiple lines.
- After selecting multiple lines type `>>` to indent them
